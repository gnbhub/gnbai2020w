## 교육 진행 계획
| 날짜 | 주제 |
|------|------|
| 1주차 (1.25) | python 및 jupyter notebook 기본 사용법 |

| 3주차 (2.8) | 인공지능의 정의와 선형 회귀 문제 |

이론을 간략하게 다룬 후 실시간으로 함께 코딩해보는 hands-on lab의 방식으로 진행합니다.<br>
